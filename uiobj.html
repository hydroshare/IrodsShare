Thoughts on the IrodsShare/Django Object Interface¶

The IrodsShare interface is designed with a specific kind of user interface in mind. The object interface in HSAccessObjects uses primitive routines in HSAlib to create “objects” that are in a sense self-describing.

The User Object¶

Instances of the HSAccessUser class describe users, including the current authenticated user. Each object is aware of the identity of the authenticated user and that user’s capabilities. For example, the following methods are written and intended to drive the contents of the user’s home page:

  • HSAccessUser.get_resources(): get a list of HSAccessResource instances describing resources to which the user has access. These do not include public resources.
  • HSAccessUser.get_groups(): get a list of groups to which the user belongs.
  • HSAccessUser.get_capabilities(): describe other capabilities via a dictionary object.

The HSAccessUser.get_capabilities() method has a return that is a dictionary object and describes the private methods to which a user has access.

Also, the primitive statistics functions

  • HSAlib.HSAccess.get_number_of_groups_of_user()
  • HSAlib.HSAccess.get_number_of_groups_owned_by_user()
  • HSAlib.HSAccess.get_number_of_resources_held_by_user()
  • HSAlib.HSAccess.get_number_of_resources_owned_by_user()

are designed for reporting statistics on the user’s home page.

The Resource Object¶

Instances of HSAccessResource describe resources to which a user has access. These instances are aware of the identity of the authenticated user. The following routines are intended for use on the resource landing page:

  • HSAccessResource.is_public(): True if resource is public.
  • HSAccessResource.is_published(): True if resource is published.
  • HSAccessResource.is_discoverable(): True if resource is discoverable.
  • HSAccessResource.is_shareable(): True if resource is shareable by non-owners.
  • HSAccessResource.get_privilege(): Returns the string privilege of the current authenticated user over the resource, which can be ‘own’, ‘rw’, ‘ro’, or ‘none’ (for public resources).
  • HSAccessResource.get_capabilities(): Returns a dictionary object describing capabilities of the current authenticated user over the resource in question.

The Group Object¶

Instances of HSAccessGroup describe groups of users in which a user is a member or which have been marked discoverable or public. The following routines are intended for use on the group landing page:

  • HSAccessGroup.is_member(): True if current authenticated user is a member of the group.
  • HSAccessGroup.is_owner(): True if current authenticated user is an owner of the group.
  • HSAccessGroup.is_public(): True if group is public; non-members can read the member list.
  • HSAccessGroup.is_discoverable(): True if group is discoverable; non-members can discover the existence of the group.
  • HSAccessGroup.is_shareable(): True if non-owners can add members to the group.
  • HSAccessGroup.get_privilege(): Returns the string privilege of the current authenticated user over the resource, which can be ‘own’, ‘rw’, ‘ro’, or ‘none’ (for public resources).
  • HSAccessGroup.get_capabilities(): Returns a dictionary object describing capabilities of the current authenticated user over the group in question.
