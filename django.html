Interfacing IrodsShare with DjangoÂ¶

In this version of IrodsShare, there are new methods in HSAlib for interfacing to Django’s access control system. Each one takes one argument that is the uuid of a resource or group.

  • For resources:
    • HSAccess.can_change_resource() returns True if the current user can change the referenced resource.
    • HSAccess.can_change_resource_flags() returns True if the current user can change the referenced resource’s flags, including ‘public’, ‘discoverable’, ‘published’, ‘immutable’, and ‘shareable’.
    • HSAccess.can_view_resource() returns True if the current user can view the resource.
    • HSAccess.can_share_resource() returns True if the current user can share the resource with others.
  • For groups:
    • HSAccess.can_change_group() returns True if the current user can change group metadata.
    • HSAccess.can_change_group_flags() returns True if the current user can change the referenced group’s flags, including ‘public’, ‘discoverable’, and ‘shareable’.
    • HSAccess.can_view_group() returns True if the current user can list members of the group.
    • HSAccess.can_share_group() returns True if the current user can invite group members.

Likewise, in HSAccessObjects, each object has exposed public methods ‘can_change’, ‘can_change_flags’, ‘can_view’, and ‘can_share’.
